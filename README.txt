*If you are updating an existing project*
- Make sure plugin is set to WSAPlayer only
- Build a fresh solution (Erase build folder before rebuilding)
- Make sure to replace the Plugin and PracticalConfig.cs file
- If you just replace the PracticalConfig.cs file without deleting your meta file, it will retain your API key.
- The PracticalConfig.cs file in the PracticalManager prefab has new fields for Version Number and Custom Tag. 
- Please fill in accordingly.
